What is Orbit?

Orbit is a minimal intent protocol designed to eliminate ambiguity. It defines how intents are structured and validated, but not how they are executed.

All Orbit implementations rely on deterministic, fail-closed validation so systems behave predictably across environments.

example_intent.json
{
  "orbit": "1.0",
  "id": "intent-abc",
  "action": "open",
  "payload": {
    "url": "https://example.com"
  }
}

Non-goals (v1)

  • No version negotiation
  • No extensible or multiple actions
  • No schema evolution
  • No execution semantics

These are intentionally deferred to v2+.
