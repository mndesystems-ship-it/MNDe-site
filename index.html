What is Orbit?

Orbit is a schema specification for single-action intents. It defines data structure and validation rules. It does not define execution logic, identity management, or authorization policies.

Orbit is a technical specification. MNDe Systems does not operate, execute, or control third-party systems that use Orbit, and assumes no responsibility for outcomes resulting from its adoption or implementation.

The reference implementations validate inputs against the v1.0 specification. Inputs deviating from the schema are rejected.

Reference implementations are provided for evaluation purposes only and do not confer certification, approval, or guarantees of correctness.

}

Non-goals (v1)

  • No version negotiation
  • No extensible or multiple actions
  • No schema evolution
  • No execution semantics

These functions are excluded from the v1 specification.
